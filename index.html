☁️ Configurar sincronización automática en la nube (Firebase)
Una vez configurado, todos los datos se sincronizan automáticamente. Sin hacer nada manual.
Paso 1
Ve a console.firebase.google.com e inicia sesión con Gmail
Paso 2
Crear proyecto → Firestore Database → "Modo de prueba"
Paso 3
Project Overview → icono </> → Registrar app web
Paso 4
Copia las 6 credenciales y pégalas en el bloque firebaseConfig al inicio del archivo HTML
// Busca esto en el HTML y rellena con tus datos:
const firebaseConfig = {
  apiKey: "AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXX",
  projectId: "mi-gimnasio-xxxxx",
  // … etc
};
📋 Rutina recomendada
☁️ Con Firebase activo
Trabajas normal. Todo se guarda solo. Puedes abrir la app en cualquier PC y los datos aparecen automáticamente.
📴 Sin internet
La app sigue funcionando con los datos locales. Al recuperar conexión, sincroniza automáticamente.
💻 PC nueva
Abres el HTML → esperas 2 segundos → todos los datos aparecen desde la nube. Sin importar archivos.
💾 Sin Firebase
Usa el respaldo manual: descarga el .json al cerrar el día y guárdalo en USB.
